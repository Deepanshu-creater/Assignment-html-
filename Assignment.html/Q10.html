HTML with Special Characters

Structure of HTML

(html): Root element of the HTML document.
(head): Contains meta-information about the document (like character encoding, title).
(title): It defines the title shown in the browser tab.
(body): It contains the content that will be displayed on the webpage.
(h1): It is used to represent a top-level heading.
(p:) To represents a paragraph.
(ul) and li: Represents an unordered list and list items, respectively.

Special Characters in HTML

Here are some commonly used special characters in HTML:

  • & represents &
  • < represents <
  • > represents >
  • © represents ©
  • ® represents ®
  • ™ represents ™

For example: "All rights reserved © 2024".
