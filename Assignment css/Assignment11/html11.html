Positioning

Static Positioning (Default)

This element is statically positioned, which is the default positioning for elements. It follows the normal document flow.

Relative Positioning

This element is relatively positioned. It is moved relative to its normal position, but it still occupies space in the normal document flow.

Absolute Positioning

This element is absolutely positioned. It is removed from the normal document flow and positioned relative to its nearest positioned ancestor or the initial containing block (if none exists).

Fixed Positioning

This element is fixed. It stays in the same position relative to the viewport even when scrolling.

Sticky Positioning

This element is sticky. It behaves like a relative element until it reaches a specific point, after which it becomes fixed.
